# calendar-ai-mcp

MCP server for AI-powered calendar and schedule management.

## Tools

- **create_event** — Create calendar events with timezone support
- **find_free_slot** — Find available time slots in a busy schedule
- **calculate_duration** — Calculate duration between two datetimes
- **timezone_convert** — Convert times between UTC offsets

## Usage

```bash
pip install mcp
python server.py
```

## Rate Limits

50 calls/day per tool (free tier).
